# Changelog

## 2026-07-08

### Added

- 新增「报账批次」：运营组把一次要交的材料自由圈选、命名、留存为一个集合，可跨报账人、跨抬头、只取某人部分条目；主界面以文件夹式批次条显示条数与缺件数。
- 批次内每条可写「批次级催办备注」（如「张三缺查验单」），与条目自身的记账备注分离。
- 批次上下文条显示合计金额与每报账人小计，并可一键导出 / 打印 / 归档整批。
- 条目支持打多个标签，并可批量打标 / 去标；高级筛选新增按标签、按有 / 无备注、按批次筛选。
- 主界面支持多栏条目列表、Shift 范围选择，并扩大复选框 / 抬头色条的选择热区，减少误打开详情。
- 身份支持编辑（姓名 / 审核人 / 学号 / 电话 / 银行信息），此前只能新增和删除。
- 设置里可把数据整体迁移到自定义位置，或恢复到系统默认位置。
- 设置新增启动默认抬头、默认列表密度偏好。

### Changed

- 设置页重构为配置中心（身份、偏好、数据位置、组件与更新、关于），移除与主界面重复的入口（批量导入 / 导入绑定包）和开发者视角的内部路径。
- 顶栏动作按主次重排：导入 / 设置归为次级，批量导入 / 新建为主操作；抬头、报账人、批次、状态、排序统一收到筛选工具条。
- 设置中的默认抬头和默认列表密度调整后会立即同步到主界面。
- 优化主界面字体、金额数字、条目选中态和首次使用提示的视觉层级。
- 数据库 schema 升级到 v2，历史库首次打开时自动补齐批次相关表。

## 2026-07-06

### Added

- 支持从文件夹或多选文件批量导入发票 PDF，并自动匹配同目录或子目录中的发票 XML。
- 支持将发票 PDF / XML 拖到主界面进入批量导入预览。
- 支持将付款截图、查验单拖到条目卡片或条目详情材料区绑定到指定条目。
- 新增首次使用提示，并在设置中提供“使用提示”入口。
- 新增拖拽中转区 `dropped/`，用于 WebView 拿不到本机路径时保存临时文件。

### Changed

- 批量导入只用发票 PDF 创建条目，XML 只作为辅助识别；付款截图和查验单不再按文件名自动绑定。
- 优化拖拽视觉反馈：空白列表区用于批量导入，条目卡片用于绑定材料，减少误拖。
- 设置页重组为常用操作、功能状态、本机数据，减少重复入口。
- 精简列表模式会收起次要信息和快捷按钮，而不只是压缩间距。
- 教育基金会条目只保留颜色和色条区分，不再额外强化显示抬头标签。

### Fixed

- 修复部分数电发票 PDF 文本流错位导致的购买方、销售方、明细和金额识别失败。
- 修复拖拽提示在 WebView 中可能不消失的问题。
- 修复 pywebview 文件对话框弃用警告，改用新版 `FileDialog` API。
- 过滤 macOS 上已知无害的 `NSSoftLinking` 输入法框架警告。
- 防止同一条目重复上传同一份附件。
- 防止将不同发票号的 PDF / XML 拖入已有条目造成误绑定。
